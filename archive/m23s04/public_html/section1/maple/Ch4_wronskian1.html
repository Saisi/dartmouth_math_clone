The wronskian

> restart;

Below are the commands used to help evaluate determinants of large matrices.

First, load the linear algebra package:

> with(linalg): 

Warning, the protected names norm and trace have been redefined and unprotected

Define a few functions - four are used in this example:

> y1:=exp(t);y2:=exp(-t);y3:=exp(2*t);y4:=exp(-2*t);

Create the wronskian matrix

> W:=matrix([[y1,y2,y3,y4],[diff(y1,t),diff(y2,t),diff(y3,t),diff(y4,t)],[diff(y1,t$2),diff(y2,t$2),diff(y3,t$2),diff(y4,t$2)],[diff(y1,t$3),diff(y2,t$3),diff(y3,t$3),diff(y4,t$3)]]);

Take the determinant:

> det(W);

Or one can use the wronskian command: the first argument is a vector of functions, the second argument is their common variable.

> wronskian([y1,y2,y3,y4],t);

> det(%);
