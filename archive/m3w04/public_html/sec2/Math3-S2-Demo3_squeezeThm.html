Class Demo for the Squeeze Theorem

January 12, 2004


   Consider any horrible function

> f := x -> sin(1/x):

    Have a look at its graph

> plot(f, -0.1 .. 0.1);

    Now squeeze it between   and

> g := x -> x^2 * f(x):

    The new function is much better

> plot([g(x), x^2, -x^2], x = -0.1 .. 0.1, thickness = 2);
