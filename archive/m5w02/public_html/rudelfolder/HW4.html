1.  Let G be the group D4 [see previous assignment for definition of D4].  Consider the imposing of a new relation:  {x2=e}.

3.    Consider the group Z12.  The subset {0,4,8} is the kernel of some map [we don't know which one].  What are the relations corresponding to this kernel?

What is the new group one would get by imposing these relations?

 

4.       We have discussed how any kernel gives rise to a set of relations, so it might be helpful for us to know how to recognize if a given set is a kernel for some map.

          Show that is Y is an element in the kernel and X is any element in the group one is mapping out of.  [That is, pretend f is a map from G to H, then let Y be an element in the kernel of f, and let X be an element in G].  Then XYX-1 is also an element in the kernel of f.  [here X-1 means "X hat"].

 

5.   Show that if f(X)=f(Y) then XY-1 is in the kernel of f.  [Once again, Y-1 means "Y hat."].
