Final Notes for Groups

 

          The final material shown in class is the existence of the correspondences among relations, functions, and kernels. 

Imposing any set of relations on a group (which may or may not already have relations upon them) gives rise to a function taking each equivalence class of words in the original ("old") group to the corresponding class in the new group.  That is, f(X)=X, where the first "X" is to be understood as a class in the old group and the second as a class in the new group .

It should be clear that any function gives rise to a kernel, in particular the kernel of the function in question.

Finally, any kernel can give rise to a new set of relations, the relations simply being that all elements in the kernels now be equal to "e."  The new group is denoted  [assuming the original group was "G" and the function f.  The final "big" result is that this group is equivalent to the image of f.  The result is known also as "The First Isomorphism Theorem."
