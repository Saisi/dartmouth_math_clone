Homework for Wednesday 2/20

NOTE: This homework is VERY good preparation for understanding the "big result" I am going to show on Wednesday.

The following is to be used for all problems:

 

S3 is the group on {a,b} with the relations {a2=e,b3=e, and ab=ba2}.

 

V is the group on {c,d} with the relations  {c2=e, d2=e, cd=dc}.

 

D4 is the group on {x,y} with the relations {x4=e, y2=e, xy=yx3}.

 

Z2 is the group on {w,z} with the relations {wz=zw}.

 

Zn is the group representing modular arithmetic modulo n (we will call the generator "1").

 

1.  How many elements do V, S3, and D4  have?

 

2.  For each of the following, decide whether the map can exist or not:

 

          i) f: S3 -> V  such that f(a)=c, f(b)=d.

          ii) f: S3 -> V such that f(a)=c, f(b)=c.

          iii) f: V -> D4  such that f(c)=x, f(d)=y.

          iv) f: D4 -> V such that f(y)=d,  f(x) =c.

          v) f: V -> S3 such that  f(c)=ab f(d)=e.

          vi) f: Z2 -> V such that f(w)=c, f(z)=d.

          vii) f: V -> Z2  such that f(c)=w and f(d)=z.

          viii) f: Z12 -> S3 such that f(1)= b.

 

3.  For each of the maps above that exist, find the kernel.

 

4.  Calculate the size of the kernel and compare it to the size of the groups involved…any observations?

 

5.  For each of the maps above that exist, find the image.

 

6.  Do you notice any relationship between the size of the image, the size of the kernel and the size of the first group in each case?

 

7.  Prove the following:  If  f  is a map that takes G to H, and X is an element in the kernel   of f,  and Y is any element in G, then f(X*Y)=f(Y).

 

8.  Prove the following:  If f  is a map from G to H, then the image of f is a subgroup of G.
