Maple

Maple is the Computer Algebra System we will be using in Math 3. Getting Started with Maple is a document that explains how to put Maple onto your computer. It also includes some sample commands to experiment with in Maple, and a note on how to get help within Maple.

The following downloadable folder contains the Math 3 Maple Files that are referred to in 'Getting Started with Maple.' The folder is a compressed file in Aladdin's ".sit" format. It can be expanded with Stuffit Expander 5.1.2 on Macs or Aladdin Expander 5.0 on PCs running Windows. These expanders are free from Aladdin. We will be putting all compressed files included in the Math 3 website into ".sit" format, so be sure that you have a copy of the appropriate expander. It should already be installed on all new machines purchased this fall at Dartmouth. If the free Aladdin expander is not on your computer, see the instructions for where to get it.

  • For both Macintosh and PC users, download this file: Math 3 Maple Files.
