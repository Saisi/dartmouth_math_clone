> "By hitting return we will load in a program capable of solving a differential equation. This is precisely the method we
discussed in lecture. The explicit equation here is dY/dt = Y^2 + sin(t), you may modify it to solve other equations. To run
the program you must specify the initial time T0, the initial value I0, the final value of interest T and the number of
increments N that you'd like to break the interval [T0,T] into.":

with(plots):


DE1:=proc(T0,I0,N,T)
local i,List,Deltat,Position,Time;
Digits:=5;
Position:=I0;
Time:=T0;
List:=[[Time,Position]];
Deltat:=(T-T0)/N;
for i from 1 to N
do
Position:=evalf(Position+Deltat*(-Position^2+5*sin(Time)));
Time:=evalf(Time+Deltat);
List:=[op(List),[Time,Position]];
od;
List;
end:

> "In order to solve the equation with time .2 corresponding to the value 4 up to time 3 with 20 increments do..":


S1:=DE1(.2,4,20,3);




> "If you don't want to view the list use a colon, like..":

S1:=DE1(.2,4,20,3):

"To plot these points first create the plot":

P1:=listplot(S1,color=red):

"Then view it":

display(P1);
