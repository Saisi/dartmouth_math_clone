The Lorenz Attractor


This is the Lorenz attractor. It is a solution to the Lorenz equations, a system of first order differential equations:

x' = -sx + sy
y' = rx - y - xz
z' = -bz + xy.

The solution shown here was computed numerically using Maple with the values b = 8/3, r = 28, s = 10 and initial conditions x(0) = 0, y(0) = -1, z(0) = 1. The curve is actually an object in three-dimensional space; what you see here is a two dimensional projection. Its true shape can be inferred from the animated graphic in the navigation bar to the left. For more detailed information on the Lorenz equations, see Section 9.8 of the course text.
