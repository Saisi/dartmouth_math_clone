  • remember to type with(DEtools); to invoke the differential equations package.

  • Solve a DE: dsolve(diff(y(x),x)=x-2*y(x), y(x));

  • Solve an initial value problem: dsolve({diff(y(x),x)=x-2*y(x), y(0)=1}, y(x));

  • Plot a slope field: DEplot(diff(y(x),x)=x-2*y(x), y(x), x=-2..2, y=-2..2);

  • Plot a slope field and a few solution curves: DEplot(diff(y(x),x)=x-2*y(x), y(x), x=-2..2, [[y(0)=1], [y(0)=-2]], y=-2..2);
