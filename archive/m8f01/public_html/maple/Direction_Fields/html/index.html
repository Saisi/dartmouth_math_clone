Direction Fields

T. R. Shemanske

First we load the differential equations library.

> with(DEtools):

Next we enter a differential equation. The function diff(y(x), x) should simply be thought of as dy/dx.

> DE := diff(y(x),x)=x+y:

Now we tell Maple to do a direction field plot of this differential equation on a sutiable domain:

> DEplot( DE ,y(x), x=-3..3, y=-3..3,color=BLUE); 

Warning, y is present as both a dependent variable and a name. Inconsistent specification of the dependent variable is deprecated, and will be removed in the next release.

Now we tell Maple to draw in solutions to the differential equation which pass through several points.

> DE := diff(y(x),x)=x+y:

> DEplot( DE , y(x), x=-3..3, y=-3..3,
[[y(0)=0], [y(0)=1], [y(0)=-1]], arrows=SMALL,
dirgrid=[30, 30], stepsize=.1,
linecolor=RED, color=BLUE); 

Warning, y is present as both a dependent variable and a name. Inconsistent specification of the dependent variable is deprecated, and will be removed in the next release.
