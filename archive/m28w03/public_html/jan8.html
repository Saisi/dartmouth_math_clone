Basic Counting Principles

Group discussion: # 5,6,8
  • Sum principle
  • product principle
Basically, we want to find the size of a set S (i.e. collection of objects):
  1. S contains possible sandwich choices given some choices of meat and bread.
  2. S contain possible choices for a banana split with three scoops of ice cream.
  3. S contain possible games in a tournament.
Def: A collection{S1,S2,...., Sn} is called a set partition of S if
    (1) S = S1 U S2 U.....U Sn
    (2) the intersection of Si and Sj is empty for any i not equal to j.

The Si's are called blocks.

Sum Principle: If we have a partition of a set S, then the size of S is the sum of the sizes of the blocks of the partition.

Product Principle: If we have a partition of a set S into m blocks, each of size n, then S has size mn.
