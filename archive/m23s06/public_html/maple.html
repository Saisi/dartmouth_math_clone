Maple Demos


Maple is available from Dartmouth's software download website.  Here's a tutorial on the Maple procedures we will be using in Math 23.  It will be periodically updated as necessary.

To download the demo files:  in Windows, right click to save link as; in OS X, hold the mouse down until the drop-down menu appears with the save link as option.  Once you have saved the file to your hard disk and opened it with Maple, make sure you activate the initial lines in the worksheet by putting the cursor at the end of each line and hitting "return."
