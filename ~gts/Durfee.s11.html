Polynomial Knots

Alan Durfee

Mount Holyoke

A polynomial knot is a smooth embedding κ: R→ R3 whose components are polynomials.   It is both an object of real  algebraic geometry as well as being an open ended topological knot, so it has  both algebraic and topological invariants.  In this talk I will give a number of examples, describe different types of equivalence and discuss the structure of the space of these knots.
