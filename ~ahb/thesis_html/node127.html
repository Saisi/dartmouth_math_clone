Smoothing

As is clear from Fig. 2.7, individual matrix elements are random quantities. Recalling the definition (2.48), the average square matrix element a certain distance from the diagonal is proportional to the desired band profile . I estimate only at equally-spaced frequency points , where integer . The value at each point is found by averaging the squared elements whose frequency difference falls within the `bin' from to . The constant bin width determines the resulting frequency resolution. This is seen to be simply smoothing via `histogram binning'. Note that the phrase `distance from the diagonal' would strictly imply an integer . This small size of jitter is due to spectral rigidity[24]. The choice of continuous over integer variable is therefore arbitrary if only features larger than the level spacing are desired. The only time my choice is important is in Section 2.3.4.

The diagonal elements are not treated as part of the band profile, and are removed before binning.
