Abstract: In 1987, Dan Gordon gave a definition of an "elliptic pseudoprime", which

is a composite number which masquerades as prime in connection with an

elliptic curve with complex multiplication. Â An elliptic Carmichael number is

a number which is simultaneously an elliptic pseudoprime for all possible curves.

In a new paper with Aaron Ekstrom and Dinesh Thakur, we show that modulo

an unproved hypothesis on the least prime in a residue class, there are infinitely

many elliptic Carmichael numbers.
